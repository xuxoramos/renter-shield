Developer API

Data is derived from publicly available government records and algorithmic analysis. It does not constitute legal findings and requires independent verification before any legal, administrative, or public action.

Authentication

Pass your token in the X-API-Key header on every request. Developer keys are granted investigator-grade access.

{{ token }}

Base URL: https://rentershield.org/api · Interactive spec: /docs

1. Property level

Look up buildings by address and retrieve a property's violation history.

MethodEndpointDescription
GET/api/investigator/address/searchSearch properties by address substring. Params: q (min 3 chars), jurisdiction, limit, offset.
GET/api/investigator/property/{bbl}Full property detail with violation records.
 
curl -H "X-API-Key: {{ token }}" \
  "https://rentershield.org/api/investigator/address/search?q=main+st&limit=10"

2. Landlord level

Search and inspect landlords (owners) and their harm scores.

MethodEndpointDescription
GET/api/investigator/landlords/searchSearch landlords across jurisdictions. Params: name, jurisdiction, min_score, limit, offset.
GET/api/investigator/landlords/{owner_id}Landlord detail: score components, SVI percentiles, confidence.
GET/api/investigator/jurisdictionsList all jurisdictions with aggregate statistics.
 
curl -H "X-API-Key: {{ token }}" \
  "https://rentershield.org/api/investigator/landlords/search?name=smith&min_score=50"

3. Combined level (property × landlord)

Query the product of the two — landlords within a jurisdiction, joined with their property portfolios and violation counts.

MethodEndpointDescription
GET/api/investigator/jurisdictions/{jurisdiction}/landlordsRanked landlords in a jurisdiction with property counts and violations. Params: min_score, min_properties, name, limit, offset.
 
curl -H "X-API-Key: {{ token }}" \
  "https://rentershield.org/api/investigator/jurisdictions/nyc/landlords?min_properties=3&limit=25"
